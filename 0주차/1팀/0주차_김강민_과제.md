문제--<br>
1부터 21사이의 홀수가 2열로 이루어진 배열을 생성하고 출력한 뒤, 
해당 배열 내에서 3의 배수인 수를True, 3의 배수가 아닌 수를 False으로 치환하여 재출력하시오




<br><br><br><br><br><br><br><br><br><br><br><br><br>



정답<br>

import numpy as np

arr1 = np.arange(1, 21, 2).reshape(-1, 2)
print(arr1,"\n")

arr2 = arr1%3==0
print(arr2)
